Política de Privacidade

Esta Política de Privacidade descreve como o aplicativo QuestController coleta, utiliza e protege as informações dos usuários.

Informações coletadas

O QuestController pode coletar informações básicas do dispositivo, como:

  • Status de conexão
  • Nível de bateria
  • Identificador técnico do dispositivo

Essas informações são utilizadas exclusivamente para o funcionamento do aplicativo e para exibição de status ao usuário.

Uso das informações

As informações coletadas não são vendidas, compartilhadas ou utilizadas para fins publicitários. Os dados são usados apenas para monitoramento e funcionamento do app.

Compartilhamento de dados

O QuestController não compartilha dados pessoais com terceiros.

Segurança

Medidas razoáveis são adotadas para proteger as informações contra acesso não autorizado.
